# Assets Folder

This folder should contain your app icons and splash screens:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

You can generate these assets using:
```bash
npx expo install expo-splash-screen
```

Or use Figma/Photoshop to create them manually.

For now, Expo will use default placeholder assets.
